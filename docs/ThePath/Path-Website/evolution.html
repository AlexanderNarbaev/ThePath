System Evolution

How The Path changes and improves over time

Core Cannot Change

The following require 90% global supermajority to modify:

  • Axiom 1: Actions have consequences
  • Axiom 2: Future is open
  • Prohibition 1: No killing (except self-defense)
  • Prohibition 2: No torture
  • Prohibition 3: No enslavement
  • Principle 1: Develop yourself
  • Principle 2: Create and help
  • Principle 3: Think long-term

Non-Core Documents

Can be updated by 2/3 majority of global community:

  • Practices and rituals
  • Community structure
  • Economic model
  • Political principles
  • Education guidelines

Local Adaptations

Each community can adapt practices to their culture while staying true to the core framework.
