Это минимальная этическая система координат для сотрудничества, основанная на рациональном анализе, теории игр и эволюционной этике. Сверхъестественные утверждения не делаются. ## Введение: как пользоваться этой библиотекой

Библиотека Пути — это тщательно подобранная коллекция текстов, которые информируют или дополняют рамку Пути. Это не требование для участников Пути, а **ресурс** для тех, кто хочет исследовать глубже.

**Это отправная точка, не канон.** Включение не означает одобрения каждой идеи в книге; исключение не означает отторжения. Путь поощряет широкое чтение и критическое мышление. Некоторые тексты представляют взгляды, которые противоречат принципам Пути — их чтение укрепляет понимание того, почему Путь выбирает свой собственный путь.

**Как ориентироваться в этой библиотеке:**

- Просматривайте по темам, чтобы найти работы, соответствующие вашим интересам
- Начните с «Рекомендуемых отправных точек» в каждом разделе, если вы новичок
- Используйте ASCII-диаграмму ниже, чтобы понять организацию
- Делайте перекрёстные ссылки с документами Пути (например, Документ 02 для аксиом, Документ 03 для этики)
