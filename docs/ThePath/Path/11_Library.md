## Introduction: How to Use This Library

The Library of the Path is a curated collection of texts that inform or complement the Path's framework. It is not a requirement for Path members, but rather a **resource** for those who want to explore further.

**This is a starting point, not a canon.** Inclusion does not mean endorsement of every idea in a book; exclusion does not mean rejection. The Path encourages reading broadly and thinking critically. Some texts present views that conflict with Path principles — reading them strengthens understanding of why the Path chooses its own path.

**How to navigate this library:**

- Browse by topic to find works relevant to your interests
- Start with "Suggested starting points" for each section if you are new to a subject
