См. [TRANSLATION.md](TRANSLATION.md) — там пошаговое руководство.

## Код и дизайн

1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/short-description`
3. Установите зависимости: `cd web && npm install`
4. Запустите dev-сервер: `npm run dev`
5. Внесите изменения
6. Проверьте сборку: `npm run build`
7. Создайте PR в ветку `main`

## Принципы

- **Дизайн-система:** используйте Tailwind CSS-токены (см. `web/src/styles/global.css`)
- **Компоненты:** маленькие, с одной ответственностью. Один файл — один компонент.
- **Контент модулей:** Markdown в `web/src/content/modules/{lang}/`. Структура задана Content Collection schema в `web/src/content/config.ts`.
- **UI-строки:** добавляйте новые строки в `web/src/i18n/ui.ts` для обоих языков.
- **Тёмная тема:** все компоненты должны работать в `[data-theme="dark"]`.
