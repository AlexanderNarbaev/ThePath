Три структурных решения:

Карточки-практики

Три карточки в ряд (как на мокапе). Мгновенно считывается. Можно кликнуть для деталей. Есть кнопка «Отметить сегодня» — сохраняется в localStorage.

Интерактивный чек-лист

Три пункта с галочками. Посетитель отмечает сделанное — сразу видит прогресс. Данные сохраняются. Более игровой подход.

Карточки + чек-лист (recommended)

Карточки объясняют практики. Ниже — мини-чеклист «Сегодня» с тремя галочками. Объяснение и действие разделены. Чек-лист — якорь для возврата.
