Что берём из брутализма

  • Моноширинный шрифт для мета-комментариев и «честных» пометок (#ОТКРЫТАЯ СИСТЕМА, версия, предупреждения)
  • Прямой язык без маркетинга: «Не религия. Не идеология. Практический инструмент.»
  • Стрелки и маркеры (▸ →) вместо декоративных элементов
  • Никаких эмодзи. Только символы.

Что берём из академического

  • Цитаты и сноски как элементы доверия (не changelog'и!)
  • «Прозрачная структура»: посетитель видит, из чего состоит система, без культа
  • Текст — главный герой. Визуал только поддерживает.
  • Факты вместо обещаний: «Основано на...», «Подтверждено...»

Принципы гибрида

1. Тёмный космос — фон

Глубокий тёмный (#0a0a14), градиенты, золотой акцент. Создаёт атмосферу «серьёзного разговора». Светлая тема остаётся для длинного чтения.

2. Типографика — два голоса

Основной голос: системный sans-serif (Inter/System UI) — тёплый, читаемый. Честный голос: моноширинный (JetBrains Mono) — для мета-комментариев, фактов, предупреждений. Два шрифта = два уровня доверия.

3. Золото — сигнал, не декор

Золотой используется только для смысловых акцентов: ключевые слова, важные кнопки, разделители смысловых блоков. Не для «красивости».

4. Никакого маркетинга

Никаких «присоединяйтесь», «уникальная возможность», «тысячи людей уже...». Только: «Вот система. Вот практики. Вот почему это работает. Решать тебе.»
